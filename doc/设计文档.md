# 程序报告生成器

Program Report Generator

## 技术实现文档（Technical Specification）

版本：v1.0
 技术路线：**JavaScript / Node.js**

------

# 1 系统目标

系统用于生成 **课程实验报告 / 实践报告**。

用户只需要：

```
填写表单
编写 Markdown
插入图片
粘贴代码
```

系统自动生成：

```
DOCX
PDF
```

并保证：

```
统一格式
自动编号
自动排版
```

------

# 2 技术架构

系统采用：

**Frontend + Node.js Backend**

架构：

```
Frontend (Web)
│
│ Markdown + Form
│
Backend API
│
Markdown Parser
│
DOCX Generator
│
PDF Generator
```

详细流程：

```
Markdown + Form
        ↓
POST /generate
        ↓
Markdown Parser
        ↓
DOCX Builder
        ↓
DOCX
        ↓
PDF Generator
        ↓
DOCX + PDF
```

# 5 API设计

## POST /generate

请求：

```
POST /generate
Content-Type: application/json
```

Body：

```
{
  "form": {
    "course": "操作系统",
    "project": "进程调度实验",
    "department": "计算机系",
    "grade": "2022",
    "name": "张三",
    "studentId": "20220001",
    "date": "2024-05-01"
  },
  "markdown": "# 实验目的\n...\n"
}
```

返回：

```
{
  "docx": "/download/report.docx",
  "pdf": "/download/report.pdf"
}
```

# 10 标题自动编号

```
numbering
```

示例：

```
1

1.1.1
```

配置：

```
const numbering = {
  config: [
    {
      reference: "heading",
      levels: [
        { level: 0 },
        { level: 1 },
        { level: 2 }
      ]
    }
  ]
}
```

# 11 代码块渲染

代码块支持：

```
文件名
行号
自动分页
```

输入示例：

```
文件：main.py

```python
print("hello")
生成结构：
```

Paragraph
 文件名

Table
 行号 | 代码

```
示例：

```javascript
new Table({
 rows: [
   new TableRow({
     children: [
       new TableCell({ children: [new Paragraph("1")] }),
       new TableCell({ children: [new Paragraph("print('hello')")] })
     ]
   })
 ]
})
```
